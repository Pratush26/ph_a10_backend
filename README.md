# BACKEND of Plate Share

This is a backend project of a food donation website named Plate share. Here, have CRUD operations for create, update, read foods, delete foods and change the food's status.

### Used Tech
- Express
- MongoDB
- Firebase

## Features
- User can create, get , delete foods data and food requests
- User can update food details
- User can update the status of food details and food requests
- For private api reuest user have to do token verificaation
- In the Critical check user have to match there token decoded email, unless their request will be rejected in the mid process
- Firebase admin used for authentication
- MongoDB used for Database
